# Collection Types - Gap Analysis

## High Priority (No Automated Tracking)

### Crafting/Gathering Logs
- Recipe completion status (Master recipes have no checkmarks in-game)
- Gathering log completion per node
- Fishing log completion per hole
- Crafted items count/achievement progress
**Why Priority**: Most requested, no existing automation, clear memory locations

### Quest Tracking
- MSQ progress/completion
- Side quest completion by zone  
- Beast tribe quest progression
- Custom delivery unlocks

### Duty/Instance Progress
- Dungeon/trial/raid first-time clears
- Extreme/Savage/Ultimate completion
- Deep dungeon floor progress (PotD/HoH/Eureka Orthos)
- Variant/Criterion dungeon completion

### Relic Weapons
- Relic quest stages per weapon
- Resistance weapon progress (Bozja)
- Manderville weapon progress
- Anima weapon step tracking

### Exploration Content
- Sightseeing log completion (web trackers exist, no memory reader)
- Aether current collection per zone
- FATE completion tracking
- Treasure map discoveries

## Medium Priority (Partial Coverage)

### Bozja/Eureka Progress
- Field notes collection (Lalachievements lists but no auto-sync)
- Lost actions inventory (Bozja/Zadnor)
- Elemental gear progress (Eureka)
- Logos actions (Eureka)

### Social/Character
- Adventurer plate unlocks
- Portrait frame unlocks (Lalachievements WIP)
- Gearset configurations
- Macro collections

### Housing/Furnishings
- Furnishing catalog ownership
- Orchestrion roll ownership (Good Memory tooltips only)
- Housing item unlocks

## Lower Priority (Niche)

### PvP Progress
- Series rewards claimed
- Frontline/CC rank
- Feast season participation

### Gold Saucer
- TT card collection (Lalachievements has this)
- GATEs completed
- Lord of Verminion unlocks
- Chocobo racing pedigree

## Initial Development Target

**Phase 1**: Crafting Recipe Completion + Gathering/Fishing Logs
- Highest user demand
- No existing memory-reading solution
- Clear use case (Master recipes lack in-game indicators)
- Well-defined memory structures
