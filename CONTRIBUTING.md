# Contributing to Akadaemia Anyder

Thank you for your interest in contributing to Akadaemia Anyder!

## Project Philosophy

This project aims to provide collection tracking through local memory reading while:
- Maintaining user privacy (all data stays local)
- Being transparent about ToS violations
- Following community tool precedents (ACT, Dalamud)
- Documenting enforcement risk accurately

## How to Contribute

### Reporting Issues
- Use GitHub Issues for bug reports
- Include FFXIV game version and OS
- Describe expected vs actual behavior
- Never share account information

### Suggesting Features
- Check existing issues first
- Reference collection type from docs/COLLECTIONS.md
- Explain use case and demand
- Consider implementation complexity

### Code Contributions
- Fork the repository
- Create feature branch (`git checkout -b feature/collection-type`)
- Follow existing code style
- Add tests for new functionality
- Update documentation
- Submit pull request
